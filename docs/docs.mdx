## Installation

<InstallScript packages="@gfazioli/mantine-compare" />

After installation import package styles at the root of your application:

```tsx
import '@gfazioli/mantine-compare/styles.css';
```

You can import styles within a layer `@layer mantine-compare` by importing `@gfazioli/mantine-compare/styles.layer.css` file.

```tsx
import '@gfazioli/mantine-compare/styles.layer.css';
```

## Usage

The `Compare` is interactive component that allows you to compare two content sections by dragging a slider. It is particularly useful for visualizing differences between two versions of data.

<Demo data={demos.configurator} />

## Styles API

`Compare` supports [Styles API](https://mantine.dev/styles/styles-api/), you can add styles to any inner element of the component with `classNames` prop. Follow [Styles API](https://mantine.dev/styles/styles-api/) documentation to learn more.

You can customize the appearance of the `Compare` component using the Styles API. This allows you to modify styles for various parts of the component, such as keys, values, brackets, and more.
